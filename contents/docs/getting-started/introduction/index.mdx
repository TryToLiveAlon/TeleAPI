## Technology & Libraries

Our API leverages cutting-edge technologies to ensure performance and flexibility:

- **Node.js** - JavaScript runtime for executing smoothly the server-side code.
- **Express.js** - It is a **fast**, **minimal**, and **flexible web framework** for Node.js, making it a popular choice for building APIs.
- **Nodemailer** - Node.js library for sending emails using SMTP.
- **xml2js** - Helps in parses XML templates into JavaScript objects.
